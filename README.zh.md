迷你便捷的本地音乐播放器，轻巧实用，专为播放本地音频而设计，支持多种常见音频格式，随时随地畅听您的音乐！

### 核心功能

- 支持本地音频文件播放
- 波形进度条显示播放进度
- 自定义播放器主题颜色
- 一键切换迷你模式
- 可在状态栏中显示并控制播放器

支持格式：MP3, AIFF, AMR, WAV, CAF, AAC, AC3, FLAC, M4R, M4A 等
